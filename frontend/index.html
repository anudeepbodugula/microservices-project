Welcome to MyProject Microservices

This project demonstrates a microservices architecture where each service is independently developed, containerized using Docker, and orchestrated via Docker Compose. Traffic is routed through a centralized NGINX reverse proxy, simulating a production-like environment for scalable and modular applications.

Technologies used: Docker, Docker Compose, NGINX, Node.js, Python (Flask), Java (Spring Boot), Go, MySQL.
